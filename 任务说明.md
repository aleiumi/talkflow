# TalkFlow - 项目任务说明

## 项目概述
TalkFlow 是一个网页版社交话题引导工具，帮助用户（尤其是社交新手）在不同社交场景下自然开启对话、延续话题、深化情感连接。

## 技术栈（待确定）
- 前端：HTML/CSS/JS（轻量级 SPA 方案待定，如 Vue3 / React / 原生）
- 存储：LocalStorage（初期）→ 后端数据库（后期）
- 部署：静态托管（Vercel / GitHub Pages）

## 核心功能模块

### 1. 话题盲盒（核心功能）
- **随机抽取**：点击大泡泡/盲盒动画，随机跳出一个话题
- **维度筛选**：
  - 场景模式：微信聊天、面对面吃饭、散步中、深夜电话
  - 关系阶段：刚加好友、初次约会、暧昧期、稳定期、见家长
  - 情绪基调：幽默风趣、治愈感性、深度思考、奇思妙想

### 2. 话术 + 思维双引导
每个话题包含三段内容：
- 【开场白】直接可用的破冰句子
- 【为什么要聊这个】背后的心理学/社交逻辑
- 【可能的后续延伸】根据对方回答的不同走向，给出应对策略

### 3. 话题实验室（互动功能）
- 用户贡献：上传自己觉得好的话题，待审核后入库
- 红黑榜：用户对话题打分，标记「必杀技」vs「冷场王」

### 4. 救急模式
- 悬浮窗 / 快速启动按钮
- 一键获取万能破冰句，适合聊天突然冷场时使用

## 非功能需求
- 响应式设计，优先适配手机端
- 加载速度快，离线可用（PWA）
- 界面风格：轻松、温暖、有活力，避免说教感
